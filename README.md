# static-prototype
**Basic frontend repo for prototypes**

### Install
```sh
$ clone https://github.com/christianheyn/static-prototype.git
$ npm install
```

### Work
```sh
$ npm run watch:scss
$ npm install watch:js
```
*or*
```sh
$ npm install watch:all
```
